Using a Parameter

Examples:

def any_square(length):
    for counter in range(4):
        forward(length)
        right(90)

any_square(50)   # draws a square with side length 50
any_square(150)  # draws a square with side length 150
any_square(60)   # draws a square with side length 60

Length is a parameter used to pass data into the function any_square.

Key Concepts

  1. The for loop is one kind of loop.
    • It uses a counter which takes on a value every time the loop iterates.
    • In the syllabus, this is called a count controlled loop.
  2. The range() function indicates how many times the statement will repeat. For example:
    • range(4) means repeat 4 times
    • range(10) means repeat 10 times
  3. In fact, range(4) generates four numbers: 0, 1, 2, 3.
    • The counter takes on each value one at a time.
    • Because there are four numbers to take the value of, there are four repeats.
