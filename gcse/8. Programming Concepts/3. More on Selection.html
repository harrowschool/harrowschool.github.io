3. More on Selection

Lesson Objectives

  • To understand how to write programs that involve more than two choices
  • To write a quiz program

Predict

What does this code do? Show what you think will be output to the screen.

def cooking():
    print("Meal planner")
    print()
    print("1. Chicken curry ")
    print("2. Veggie lasagne")
    print("3. Burger and salad")
    print()
    print("Which of these meals is your favourite? (1, 2 or 3) ")
    answer = input()
    if answer == "1":
        print("Chicken curry coming up")
    elif answer == "2":
        print("Veggie lasagne coming up")
    else:
        print("Burger and salad coming up")
    print("Enjoy!")

cooking()

Run and Modify

On your own:

  • Download the cooking program and check that it does what you thought it would
  • Change the meals to meals that you like
  • Add “None of these as an option”

Then:

  • Make a new copy of your code and secretly add 5 errors to it
  • Swap programs with your partner
  • See if you can find and correct the 5 errors
  • Write comments in the program where you found them

Make

Using your Python skills so far you are going to create a 5-question quiz on Geography.

Steps:

  1. Write down 5 questions and 3 possible answers to each
  2. Write your program
  3. Test all the different options

You will be scored on:

  • Whether it works
  • Quality of your questions
  • How well it is presented (spacing, grammar etc.)

Review

Swap with someone else and ask them to test your quiz.
